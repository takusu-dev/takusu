ポイント

  • 1 つの assistant ターンは 1 つの foldable で、中に thinking / tool / thinking ... の履歴を並べる。
  • 「考え中」chip と thinking カードのドットは、進行中の thinking だけ動く。
  • thinking の turn を抜けたら（ツール呼び出しや回答に移ったら）アニメーションは止まる。
  • foldable が画面の 3/5 を超える高さになると、下側にも「畳む」トグルが現れる。
  • 最終回答は foldable の外に表示し、thinking は折りたたみ可能な履歴として残す。

この HTML はあくまで mock です。実装時は mobile/src/views/AgentView.tsx の thinking / tool context 部分を置き換えてください。
