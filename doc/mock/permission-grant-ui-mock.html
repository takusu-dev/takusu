ポイント

  • *:* を画面上部のマスタートグルにして、個別権限との関係を明確にする。
  • 個別トグルはリソース別（タスク / 習慣 / スケジュール / 記憶 / スキル）にグループ化する。
  • 削除など破壊的な操作は色で目立たせる。
  • カテゴリヘッダーですべて ON / すべて OFF を一括切り替えられる。
  • 検索ですぐに対象権限にたどり着ける。
  • マスター ON 時は個別トグルを disabled にし、すべて自動承認されることを示す。

この HTML はあくまで mock です。実装時は mobile/src/components/PermissionsEditor.tsx を置き換えてください。
