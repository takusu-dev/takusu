# ホーム画面

ホーム画面はタスクを時系列順に表示するメイン画面です。

## 構成

- **上段**: ハンバーガーメニュー、検索ボタン、同期ボタン
- **中段**: タスクカードのリスト
- **下段**: タスク追加ボタン、タスク実行開始/Done ボタン

## タスクカード

- 左側: 開始/終了時刻
- 中央: タイトル
- 背景色: `abandonability` に応じて変化
- 右下: 所要時間（avg, σ）

## 操作

- **タップ**: タスク詳細画面へ
- **長押し**: 選択モード
- **右スライド**: Done
- **左スライド**: 削除
- **長押し＋ドラッグ**: タスクの順序を調整

## グルーピング

- `pending` のタスクは上部に表示
- 日付が変わるごとに区切り線が入る
- 過去のタスクは上に引っ張ると表示
