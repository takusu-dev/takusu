# タスク詳細

タスクをタップすると表示される詳細画面です。

## 表示項目

- タイトル
- 開始時刻 → 終了時刻
- 並列タスク（並列可能な場合）
- 所要時間（avg, σ）
- `abandonability`（5 段階スライダー、中央が 0.5）
- 習慣リンク（habit から生成された場合）
- 説明文
- 並列設定
- 依存関係グラフ（関連タスクのみ）

## 編集

各項目をタップして編集できます。変更は `takusu-local` または Worker に保存されます。

## 依存グラフ

依存関係を有向グラフで表示します。完了済みタスクは灰色で表示されます。
