# ユーザー入力ツール

## `correct_asr`

音声認識（ASR）の結果が曖昧な場合に、ユーザーに確認を求めます。文脈から判断できない固有名詞、同音異義語、数値、日時、曜日、アクション対象を扱います。

```json
{
  "questions": [
    {
      "text": "こうれい",
      "for": "「高麗」か「校令」か曖昧です。どちらの漢字ですか？"
    }
  ]
}
```

| パラメータ | 型 | 説明 |
|------------|-----|------|
| `questions` | array | 質問リスト |

`questions` の各要素:

| フィールド | 型 | 説明 |
|------------|-----|------|
| `text` | string | ASR で認識されたテキスト |
| `for` | string | 何のために使い、なぜ曖昧か |

## 使い方

- 明らかな ASR エラーは文脈から推測して修正する。
- 固有名詞・同音異義語・数値・日時・曜日・ターゲットが曖昧な場合に使用する。
- 複数の質問は 1 回の呼び出しにまとめる。
- 不要な確認を増やさないよう注意する。
